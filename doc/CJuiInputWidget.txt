
*CJuiInputWidget*   >> |CJuiWidget| >> |CWidget| >> |CBaseController| >> |CComponent|
=================

 CJuiInputWidget is the base class for JUI widgets that can collect user
 input.

*CJuiInputWidget::resolveNameID*				 protected
>
 resolveNameID($nameProperty='name', $attributeProperty='attribute')
<
(array) name and ID of the input: array('name','id').

 Resolves name and ID of the input. Source property of the name and/or source
 property of the attribute could be customized by specifying first and/or
 second parameter accordingly.
    [nameProperty] (string) class property name which holds element name to be
    used. This parameter is available since 1.1.14.
    [attributeProperty] (string) class property name which holds model
    attribute name to be used. This parameter is available since 1.1.14.
